**Document Types** define the supporting documents that loan officers and applicants must upload when submitting a credit application. Each document type you configure here appears in the **Required Documents** section of the credit application form.

Navigate here via **Home → Document Types**.

---

![Document Types list](/images/document-types.png)

## Page Overview

The list shows all configured document types for your organisation. An **Add Document Type** button in the top-right opens the creation form. Each row shows the document's name, description, whether it is required, its status, and inline action icons.

<Aside type="note">
  Document types marked as **Required** must be uploaded by the loan officer before a credit application can be submitted. Optional documents can be uploaded but are not enforced at submission.
</Aside>

---

## Table Columns

| Column | Description |
|---|---|
| **Name** | The display name of the document type (e.g. *Bank / Mobile Money Statement for 6 months*). Click to open the [Document Type Detail](/settings/scoring/document-types/view-type) page. |
| **Description** | A short explanation of what the document should contain. |
| **Required** | Whether the document must be uploaded before the application can be submitted. Shows a red **Required** badge if mandatory. |
| **Status** | Whether the document type is **Active** (shown on application forms) or inactive (hidden from forms). |
| **Actions** | Inline icons: ✏️ Edit, 🗑 Delete. |

---

## Search & Export

Use the **search bar** to filter document types by name. The **Export** button supports the following formats:

- **PDF** — for documentation or compliance records
- **Excel** — `.xlsx` for analysis or record-keeping
- **CSV** — plain-text for importing into other tools
- **Copy to Clipboard** — for quickly pasting into another application

Use **Column Visibility** to show or hide specific columns.
